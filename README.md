### Pokemon Hangman Game
  > This hangman game challenges the user to guess the pokemon based on their type
  
  > ![pokemon hangman](https://cloud.githubusercontent.com/assets/18745344/20450958/7190ac12-ada9-11e6-90d8-77aac7814670.gif)
  
  
  
## How To Play
  > 1. Each blank corresponds to a letter in the Pokemon's name
  > 2. The user must type what he/she thinks the letter is in the name
  > 3. For every word that the user gets correct, 1 point is added to the Wins counter
  > 4. If a user gets a letter wrong 1 Chance is subtracted
  > 5. If the Chance counter reaches zero, the Wins counter resets to 0
